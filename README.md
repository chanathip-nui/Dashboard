แอปพลิเคชันเว็บแบบ Interactive ที่สร้างขึ้นด้วย **Dash** และ **Plotly**

สำหรับแสดงผลแนวโน้มราคาปิด (Closing Price) ของสินทรัพย์ทางการเงินหลัก 3 ประเภท ได้แก่ หุ้นทองคำ (Gold Stock), น้ำมันดิบ (Crude Oil) และดัชนี S&P 500

ก่อนรันแอปพลิเคชัน โปรดตรวจสอบให้แน่ใจว่าได้ติดตั้ง Python (แนะนำเวอร์ชัน 3.7 ขึ้นไป) และไลบรารีที่จำเป็นแล้ว
ติดตั้งไลบรารีทั้งหมดผ่านคำสั่ง:
'''bash
pip install dash pandas plotly dash-ag-grid'''

ข้อมูลที่ต้องใช้ (Data Files)
โปรเจกต์นี้จำเป็นต้องใช้ไฟล์ CSV 3 ไฟล์ ซึ่งต้องวางไว้ในโฟลเดอร์เดียวกันกับไฟล์สคริปต์ Python (.py):
1. goldstock_v1.csv - ข้อมูลราคาหุ้นทองคำ
2. Crude_Oil_Data.csv - ข้อมูลราคาน้ำมันดิบ
3. SP500.csv - ข้อมูลดัชนี S&P 500

วิธีการใช้งาน (How to Run)
1. เปิด Terminal หรือ Command Prompt

2. นำทาง (Navigate) ไปยังโฟลเดอร์ที่เก็บไฟล์โค้ดและไฟล์ CSV ของโปรเจกต์

3. รันสคริปต์ Python ด้วยคำสั่ง (สมมติว่าไฟล์ชื่อ app.py):

```bash
python app.py
4. เปิดเว็บเบราว์เซอร์และเข้าไปที่ http://127.0.0.1:8050/ เพื่อใช้งาน Dashboard
