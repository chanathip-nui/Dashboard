แอปพลิเคชันเว็บแบบ Interactive ที่สร้างขึ้นด้วย **Dash** และ **Plotly**

สำหรับแสดงผลแนวโน้มราคาปิด (Closing Price) ของสินทรัพย์ทางการเงินหลัก 3 ประเภท ได้แก่ หุ้นทองคำ (Gold Stock), น้ำมันดิบ (Crude Oil) และดัชนี S&P 500

ก่อนรันแอปพลิเคชัน โปรดตรวจสอบให้แน่ใจว่าได้ติดตั้ง Python (แนะนำเวอร์ชัน 3.7 ขึ้นไป) และไลบรารีที่จำเป็นแล้ว
ติดตั้งไลบรารีทั้งหมดผ่านคำสั่ง:
```bash
pip install dash pandas plotly dash-ag-grid
